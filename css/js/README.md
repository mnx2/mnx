# MNX Landing Page

صفحة هبوط احترافية لمشروع MNX - بطاقات NFC الذكية.

## المميزات:
- تصميم عصري ومبسط.
- دعم متعدد الأجهزة (Responsive).
- تحسين SEO.
- تفاعلية باستخدام JavaScript.

## كيفية الاستخدام:
1. قم بتنزيل المشروع كملف ZIP.
2. افتح الملف `index.html` في المتصفح.
3. قم بتعديل المحتوى حسب احتياجاتك.
